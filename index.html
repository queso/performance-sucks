Optimizing Mysql


        Episode Load (0.4ms)  SELECT `episodes`.* FROM `episodes` WHERE `episodes`.`slug` = 'test' LIMIT 1
        Episode Load (1.7ms)  SELECT `episodes`.* FROM `episodes` WHERE (number <> 2) ORDER BY number DESC LIMIT 1
